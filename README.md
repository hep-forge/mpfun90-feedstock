Feedstock for [mpfun90](https://github.com/APN-Pucky/mpfun90) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

MPFUN90: arbitrary precision Fortran computation package

## Architectures

State per architecture: see the platforms badge above or [anaconda.org/hep-forge/mpfun90](https://anaconda.org/hep-forge/mpfun90).


## Install

```bash
conda install -c hep-forge -c conda-forge mpfun90
```
